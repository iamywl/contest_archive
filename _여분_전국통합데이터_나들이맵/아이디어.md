# 나들이맵 (Outing Map)

> 아이 데리고 나갈 때 필요한 모든 정보를 한 지도에

## 한 줄 요약
공영주차장·공원·공중화장실·기저귀교환대·수유실·어린이도서관을
"유아 동반" 필터 하나로 묶어서 가족 나들이 동선을 자동 추천.

## 활용 공공데이터
- 공영주차장 실시간 주차면 (전국)
- 공중화장실 위치 (기저귀교환대·장애인화장실 필터)
- 수유실 위치 (여성가족부)
- 어린이도서관·작은도서관
- 공원·키즈카페·박물관

## 차별점
- **"유아동반" 원클릭 필터**: 모든 POI에 유아 친화도 태그
- **동선 자동 생성**: 주차 → 화장실 → 놀이터 → 식사 → 복귀 루트
- **혼잡도 예측**: 토·일 오후 2시 공원 혼잡도 시계열 예측

## 타깃
0~7세 자녀를 둔 30~40대 부모
