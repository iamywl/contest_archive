# 쉬운말 (Easy-Read)

> 발달장애인·저문해인을 위한 공공문서 쉬운말 변환기

## 한 줄 요약
사진 찍거나 PDF 업로드하면, 행정 안내문·계약서·병원 처방 설명서를
**쉬운 문장 + 그림 픽토그램 + 음성** 3채널로 변환해주는 앱.

## 핵심 기술
- OCR + 문서 구조 인식 (표·도장·서명란)
- LLM 기반 쉬운말 요약 (한국장애인개발원 「쉬운 글 가이드라인」 준수)
- 픽토그램 자동 매칭 (ARASAAC·AAC 심볼셋)
- 한국어 TTS + 한 줄씩 하이라이트 리딩

## 차별점
- **"다시 물어보기"**: 특정 단어 탭 → 해당 단어만 더 쉽게 재설명
- **중요도 하이라이트**: 금액·날짜·벌칙·서명 위치 자동 강조
- **보호자 2차 확인**: 가족·사회복지사에게 요약 공유

## 타깃
발달장애인 약 25만 명 + 저문해 노인 + 한글 미숙 다문화 가정
