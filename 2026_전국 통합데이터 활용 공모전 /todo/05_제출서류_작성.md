## 3. [붙임2] 참가서약서 — 자필 서명

1. 한글 파일 인쇄 (A4)
2. **서약자 성명** 란에 손글씨 기재 + **서명 또는 도장**
3. 스캔 또는 스마트폰 스캔 앱(네이버 Vflat, Adobe Scan)으로 PDF 화
4. 파일명 `[붙임2] 참가서약서_서명본.pdf`

### 유의 사항

- 서약 내용을 미리 읽어 숙지할 것:
  > 제출한 결과물은 반환되지 않음 / 사용권은 제작자와 주최기관에 귀속 / 지재권 침해 작품은 심사 제외 등

## 5. [붙임4] 서비스 개발 기획서 — 한글 파일로 포팅

### 5.1 내용 원본

[docs/제안서.md](../docs/제안서.md) 에 **공고 지정 6개 섹션 순서** 로 이미 작성되어 있음.
참고문헌 16건까지 포함.

### 5.2 한글 파일로 옮기는 절차

1. `[붙임4] 서비스 개발 기획서.hwpx` 를 한글프로그램에서 열기
2. VS Code 등에서 `docs/제안서.md` 열기
3. 섹션별로 복사 → 한글에 붙여넣기. 표는 **표로 직접 작성**(Markdown 파이프 그대로 붙여넣지 말 것)
4. 스크린샷 삽입 — `docs/screenshots/` 의 주요 4~6장을 선택해 기획서 본문·부록에 배치
5. 참고문헌 섹션은 마지막 페이지에 그대로 포팅

### 5.3 분량 제약 ("10페이지 이내")

현재 md 원본이 A4 환산 약 12~14 페이지이므로 다음 작업으로 축약:

- **표를 줄이고 핵심만**: §3.1 비교표, §5.1 기술스택, §6.3 정량목표 3개만 남기고 나머지는 문단으로
- **폰트 크기 10pt, 줄간격 160%** 로 조정
- **참고문헌** 은 각주 처리하거나 9pt 로 축소

### 5.4 부록 필수 기재

```
부록. 시연 URL 및 QR
- 배포 URL: https://chekcin.vercel.app
- 시연 영상: https://youtu.be/XXXXXXX
- QR 코드: [이미지 삽입]
```
