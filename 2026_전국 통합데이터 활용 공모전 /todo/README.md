# 📋 사용자 수행 체크리스트

책크인 MVP 는 완성됐으나, **제출·배포 단계에서 사용자 본인이 수행해야 하는 작업**이 3가지 남아 있습니다.
각 작업별 상세 가이드는 아래 문서를 순서대로 따라가면 됩니다.

| 3 | **시연 영상 녹화** (1~2분) | 30분~1시간 | [03_시연영상_녹화.md](03_시연영상_녹화.md) |
| 4 | **(선택) OpenAI·Anthropic 키 연동으로 AI 코치 고도화** | 2시간 | [04_AI_API_연동.md](04_AI_API_연동.md) |
| 5 | **공모 서류(hwpx)에 내용 복사·서명** | 30분 | [05_제출서류_작성.md](05_제출서류_작성.md) |

---

## 의존 관계

```
[1. API 키] ─┐
             ├─▶ [2. Vercel 배포] ─▶ [3. 시연 영상]
[4. AI 키] ──┘                              │
                                            ▼
                                   [5. 제출서류]
```

- 1·4는 독립적 — 병렬 진행 가능.
- 2는 1·4의 키를 Vercel 환경변수에 주입해야 하므로 이후에 실행.
- 3은 2에서 나온 배포 URL로 브라우저를 녹화.
- 5는 2의 URL·QR·시연 영상 링크를 기획서(hwpx) 부록에 기재하며 완료.

## 완료 기준 (제출 직전 체크)

- [ ] `.env.local` 에 `DATA_GO_KR_KEY` 설정, `pnpm dev` 로 실 데이터 확인
- [ ] `pnpm build` 다시 통과
- [ ] Vercel Production URL 발급 (예: `https://chekcin.vercel.app`)
- [ ] 모바일에서 해당 URL 접속 → 모든 화면 동작 확인
- [ ] URL → QR 코드 생성 (`docs/qr.png` 로 저장)
- [ ] 시연 영상 mp4 녹화 (1~2분)
- [ ] [붙임4] 기획서 부록에 URL·QR·영상 링크 기재
- [ ] [붙임1] 참가신청서 팀원 정보 직접 작성
- [ ] [붙임2] 서약서 자필 서명
- [ ] [붙임3] 개인정보 동의서 자필 서명
- [ ] 이메일 `opendata@klid.or.kr` 로 제출 일체 송부
