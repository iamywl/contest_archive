06 · CORE FEATURES

핵심 기능 5종 상세

각 기능은 독립적으로도, 서로 연계되어도 작동하도록 설계됩니다.

01
🚨 EMERGENCY

사고 즉시 증거 보전 — 원터치 SOS

사고 발생 시 가장 먼저 사라지는 것은 ‘증거’다. 회사가 CCTV를 지우기 전에 모든 것을 자동으로 기록한다.

  • 홈 화면 SOS 버튼 1회 → 카메라·녹음·GPS 동시 활성화
  • 모든 미디어에 SHA-256 해시 + 타임스탬프 → 법정 증거 효력 (위변조 불가)
  • 지정 비상연락망(가족·이주노동자센터·영사관)으로 자동 알림 SMS
  • 네트워크 단절 시 로컬 저장 후 복구 시 자동 업로드
02
⭐ KILLER FEATURE · AI

AI 서류 자동 생성 — 모국어 음성 → 한국어 정형 양식

이 사업의 핵심 차별점. 음성으로 진술만 하면 법적 효력이 있는 한국어 서류가 자동으로 만들어진다.

  • AI 챗봇이 모국어로 6하원칙 인터뷰 진행 (Whisper STT 기반)
  • Claude Sonnet 4.6이 진술을 분석 → 산재요양급여신청서 / 임금체불 진정서 / 진술서 자동 작성
  • 근로복지공단·고용노동부 공식 양식 100% 준수 (PDF·HWP)
  • 전자서명 후 ‘근로복지공단 토탈서비스’로 원클릭 제출
03
🗺️ MATCHING

다국어 전문가·기관 매칭 — 위치·언어 기반

모국어가 통하는 노무사·산재 지정병원·이주노동자센터를 위치 기반으로 매칭한다.

  • 16개국어 DB 구축: 노무사 · 변호사 · 산재병원 · 영사관 · 이주민센터
  • 전화 통화 시 GPT-4o Realtime 기반 실시간 AI 통역 중계
  • 1350(고용부) 콜센터 직통 + 통역 자동 호출
  • 예약·상담료 견적·후기 시스템
04
🤝 COMMUNITY

익명 동료 자조 + 사례 DB

같은 처지의 노동자들이 어떻게 권리를 회복했는지 본다. 악덕 회사·브로커는 익명 신고로 걸러낸다.

  • 국적·산업군·사고 유형별 익명 사례 검색 ("프레스 사고로 산재 인정받은 베트남인 사례")
  • 회사 평판 + 임금체불 이력 블랙리스트 (집단 지성)
  • 같은 국적 멘토 노동자와 1:1 연결
  • 모든 글은 모국어 ↔ 한국어 자동 번역
05
⏰ AWARENESS

권리 만료 시효 자동 카운트다운

아는 만큼만 권리다. 모르는 시효는 가만히 사라진다. WorkBuddy가 매일 알려준다.

  • 산재(3년) · 임금체불(3년) · 퇴직금(3년) · 부당해고(3개월) 자동 카운트
  • D-30, D-7, D-1 시점 다중 알림 (앱 + SMS + 카카오)
  • 본인 비자 만료일과 연동 — "체류 종료 전 반드시 처리할 권리" 자동 추출
