# 휠체어맵 (WheelMap KR)

> 휠체어 사용자를 위한 배리어프리 경로/시설 지도

## 한 줄 요약
경사로·엘리베이터·장애인 화장실·저상버스·장애인 주차장을 통합해
"턱이 5cm 이상인 경로는 피하는" 진짜 배리어프리 내비게이션.

## 핵심 기술
- 저상버스 실시간 API (국토부 TAGO)
- 장애인 편의시설 DB (보건복지부 BF 인증시설)
- 사용자 제보형 POI (구배·폭·문턱) 크라우드소싱
- 경로 가중치 그래프 (휠체어 통행 불가 엣지 제거)

## 차별점
- **"내 휠체어" 프로파일**: 수동/전동/폭/회전반경별 필터 다름
- **사진 제보 1탭**: 고장난 엘리베이터 사진 올리면 24시간 경로 제외
- **지자체 연계**: 제보 많은 곳에 시설 개선 리포트 자동 발송

## 타깃
휠체어 사용자 약 30만 명 + 유모차 보호자 + 노인 실버카트 사용자
