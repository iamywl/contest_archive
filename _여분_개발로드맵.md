```bash
ollama pull llama3.3:70b-instruct-q4_K_M      # ~40GB, 5개 프로젝트 공용

ollama pull gemma3:27b-instruct-q4_K_M        # ~16GB, 모두대화
ollama pull qwen2.5:7b-instruct-q4_K_M        # ~4.7GB, 안심귀가 긴급 분류
# 이미 보유: devstral-2:123b (74GB), nomic-embed-text

# STT (whisper.cpp 또는 MLX Whisper)
brew install whisper-cpp
# 또는: pip install mlx-whisper && mlx_whisper --model mlx-community/whisper-large-v3-mlx

# TTS (Kokoro — 한국어 경량)
pip install kokoro                             # https://github.com/hexgrad/kokoro

# 비전 / OCR
pip install ultralytics paddleocr mediapipe
```

**총 다운로드 예상**: ~90GB (기존 74GB + 신규 ~86GB 중복 제외 후). 128GB RAM 환경에서 단일/동시 구동 모두 가능.

### 8.3 동시 실행 메모리 예산 (M4 Max 128GB 기준)

| 동시 구동 조합 | 예상 RAM | 가능 여부 |
|---|---|---|
| llama3.3:70b 단독 | ~45GB | ✅ |
| devstral-2:123b 단독 | ~80GB | ✅ (여유 적음, 단독 구동 권장) |
| qwen2.5:32b + whisper large-v3 + MediaPipe | ~25GB | ✅ |
| 시연 조합: 실버친구(llama3.3:70b) + Streamlit × 3 | ~55GB | ✅ |
| **devstral-2:123b + llama3.3:70b 동시** | ~125GB | ⚠️ **금지** (OOM 위험) |

→ 심사 시연 시나리오는 **한 번에 1~2개 LLM만 로드**. 프로젝트 전환 시 `ollama stop <model>` 로 메모리 회수.
