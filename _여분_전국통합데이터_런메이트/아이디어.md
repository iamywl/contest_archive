# 런메이트 (Run Mate)

> 오늘 뛰어도 되는 날인지, 어디서 뛸지 알려주는 러닝 동반자

## 한 줄 요약
대기질·기온·습도 + 공원·하천·둘레길 + 공영자전거 반납소를 묶어
"지금 30분 러닝 OK 코스" 를 실시간 추천하는 시민 운동 앱.

## 활용 공공데이터
- 에어코리아 실시간 대기질
- 기상청 동네예보 (체감온도·자외선)
- 공원·하천변 자전거길·둘레길
- 공영자전거 대여소 실시간 재고
- 음수대·공공화장실 위치

## 차별점
- **러닝 신호등**: 초록/노랑/빨강 + 이유 한 줄
- **페이스별 코스 추천**: 초보(3km) / 중급(5km) / 고급(10km+)
- **"러닝 후 자전거 귀가" 모드**: 편도 코스 + 자전거 반납소 자동 매칭

## 타깃
20~40대 생활체육 러너 + 러닝 크루 운영자
